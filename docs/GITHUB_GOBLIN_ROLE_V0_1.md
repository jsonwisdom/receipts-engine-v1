# GitHub Goblin Role V0.1

Authority: false

## Function
Replay witness.

## Allowed Actions
- Store exact artifact bytes
- Preserve commit history
- Run replay workflows
- Emit logs
- Open divergence issues
- Attach hashes to manifests
- Mark fixtures as match_confirmed or divergence_detected

## Forbidden Actions
- Declare truth
- Override schema rules
- Silently repair artifacts
- Promote governance without manifest evidence
